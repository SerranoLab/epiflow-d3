About EpiFlow ▼

EpiFlow D3 — Spectral flow cytometry analysis platform for multiparametric histone H3 post-translational modification (PTM) profiling.

Statistical approach: Biological replicates are the unit of inference. Linear mixed models (LMM) account for cell-level nesting within replicates. Cell-level tests (KS, Wilcoxon, chi-square) are flagged as exploratory. Effect sizes (Cohen's d, Cliff's δ, Cramér's V) are prioritized over p-values. BH correction is applied for multiple comparisons.

ML caution: RF, GBM, and LDA train/test splits are at the cell level. Accuracy may be optimistic due to replicate leakage. Leave-one-replicate-out CV is recommended for rigorous validation.

Statistical Analysis

LMM accounts for nested data structure (cells within replicates). Replicate = random effect to avoid pseudoreplication. Stratify by Identity/Cell Cycle to find population-specific effects.

Distribution divergence heatmap

Metric:
EMD / pooled IQR (default) is the recommended metric — Earth Mover's Distance integrates the gap between ECDFs across the whole distribution, capturing both location and shape changes (Orlova et al., PLOS ONE 2016). It catches multimodal redistributions that the Kolmogorov-Smirnov D-statistic, which measures only the maximum ECDF gap, can underestimate. KS is kept available for backward comparison with previous analyses. Star annotations use BH-adjusted LMM p-values for the EMD view and BH-adjusted KS p-values for the KS view.

Correlation Analysis

Global computes one correlation matrix across all cells. Per-Group + Differential computes separate matrices per genotype, then applies Fisher z-transform to test whether marker-marker correlations differ significantly between groups. By default, the Fisher z uses replicate-level N for proper inference. Check "Use cells as replicates" only if your experiment lacks biological replicates. Disrupted co-regulation (Δr with p<0.05) indicates chromatin regulator loss affects the coordinated deposition of marks.

Positivity / GMM

Fits a 2-component Gaussian Mixture Model (GMM) to identify positive (high) vs negative (low) subpopulations for each H3-PTM mark. The threshold is set at the GMM crossover point. Fraction-positive is compared between groups using a replicate-level t-test (primary inference) and Fisher's exact test (cell-level, exploratory). Distribution shifts are quantified by Earth Mover's Distance (Wasserstein-1), which captures both location and shape changes — including multimodal redistribution that the KS test misses (Orlova et al., PLOS ONE 2016). Leave "Manual threshold" blank for automatic GMM-based detection.

UMAP

n_neighbors: low = local structure (tight clusters), high = global topology. min_dist: low = dense clusters, high = spread out. ℹ️ Hover for details.
UMAP preserves local neighborhoods to reveal cell population structure. Select a marker from the color dropdown to see fluorescence intensity (FeaturePlot). Toggle Split to compare genotypes side-by-side on the same embedding. Re-coloring uses cached data — no need to re-run.

Clustering

Clusters cells by H3-PTM profiles on a UMAP embedding. K-Means: specify k. Hierarchical: Ward linkage. Louvain/Leiden: graph-based, resolution controls granularity. Use Elbow Scan to find optimal k.
